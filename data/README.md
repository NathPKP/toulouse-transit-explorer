# data/

This directory holds all project data. Most of it is **gitignored** and must be generated locally by running the pipeline scripts.

## Structure

```
data/
├── raw/               ← GTFS zip (gitignored — download manually)
│   └── tisseo_gtfs_v2.zip
├── parquet/           ← Generated by tisseo_ingest.py (gitignored)
│   ├── stops.geoparquet
│   ├── routes.parquet
│   ├── trips.parquet
│   ├── stop_times.parquet
│   ├── shapes.geoparquet
│   ├── calendar_dates.parquet
│   └── _meta.json
└── geo/               ← GeoJSON + indicator parquets (partially tracked)
    ├── iris_toulouse.geojson          ← generated by tisseo_prepare_iris.py
    ├── quartiers_indicators.parquet   ← generated by tisseo_indicators.py
    └── hourly_indicators.parquet      ← generated by tisseo_indicators.py
```

## Setup

### 1. Download GTFS data

Get the latest Tisséo GTFS feed from:  
https://data.toulouse-metropole.fr/explore/dataset/tisseo-gtfs/

Save it as `data/raw/tisseo_gtfs_v2.zip`.

### 2. Run the pipeline

```bash
# Ingest GTFS → Parquet
python scripts/tisseo_ingest.py

# (Optional) Validate ingested data
python scripts/tisseo_check.py

# Download IRIS district boundaries
python scripts/tisseo_prepare_iris.py

# Compute coverage indicators
python scripts/tisseo_indicators.py
```

After these steps, launch the app with `streamlit run app.py`.
